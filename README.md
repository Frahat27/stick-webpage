# stick-webpage
Plataforma web oficial de STICK – Sistema de alineadores invisibles con seguimiento y rediseño clínico. Incluye sección para pacientes y ortodoncistas.
Sitio web multipágina de STICK, marca de alineadores invisibles con modelo híbrido B2C + B2B.
La web comunica seriedad, transparencia y control clínico real del tratamiento, diferenciándose de modelos cerrados tradicionales.
Incluye arquitectura SEO fuerte (alineadores invisibles + búsqueda local) y dos universos principales: Pacientes y Ortodoncistas.
Optimizada para performance, escalabilidad y generación de confianza.
